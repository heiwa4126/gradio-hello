# gradio-hello

[Gradio](https://www.gradio.app/) の練習。

## Installation and run

```console
pip install gradio-hello
gradio-hello &
```

## 開発

Hatch(1.10) で書いてある。Vite 風の run-scripts が少し。

```sh
# 開発モード (auto reload)
hatch run dev
# プリビューモード (auto reload しない)
hatch run preview
```
